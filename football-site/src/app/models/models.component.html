Machine Learning Models

First and foremost, the models below are all results of my working through this book by Aurélien Géron. A lot of these models are not that great and are all a work in progress, especially the pregame models. The games below are from the 2020-2021 Divisional Round Playoffs.

Information on training/test data here

Optimized

The optimized column indicates whether or not the model was optimized in any way, whether it be through a grid search, randomized grid search, dropping of unimportant features, etc. In the future I plan on detailing each specific model with the params and steps taken to produce the model.

Pregame Prediction

This column indicates whether the predictions are made using solely season stats, rather than stats for the predicted game after the fact. These predictions are usually much worse than post game predictions, which is to be expected.
