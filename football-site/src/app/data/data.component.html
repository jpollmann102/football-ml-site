Feature Correlations

Below are the feature correlation values for both the home score and away score labels. Features closer to 1.0 are higher positive correlation, while features closer to -1.0 have higher negative correlation. Values around 0 tend to have little to no correlation. Exploring feature correlations is a great way to drop features for your models in an attempt to improve accuracy.

A clarification on the naming of some features. Features with forced are defensive stats, meaning that home_forced_int_per_game indicates the number of interceptions the home team's defense get per game. On the flip side, home_int_per_game indicates the home team's thrown intereceptions per game.
